杜甫

发表于 2022-01-05 更新于 2022-01-20 分类于 读书笔记

读完此书，花了4h。和狗同学讨论，他说你4h能读出些什么？是一个一个字读的，还是跳读的。我说跳读的，按照主线然后找证据这种方式来读的。基本上了解大概。

那你说说李白和杜甫什么关系？

我说，李白是杜甫的前辈。杜甫眼中，他崇拜且珍视与李白的友谊；可是在李白眼中，似乎没有太care这个朋友。作者洪业说，这是因为，李白是浪漫主义者，不太注重人与人之间的关系；杜甫是现实主义者，对人很重感情。你看看他对对家人、对朋友、对国家的感情，就知道了。

狗同学瞅了我一眼说，我明白了。你就是李白，我就是杜甫。

有本事

发表于 2021-12-27 更新于 2022-01-17 分类于 读书笔记

飞往海南的飞机上读了一半，提示要有会员才能再读。网上耗费时间寻找免费电子书，不得。又按捺不住看他文字的冲动。于是买了会员，继续读完。

总体，冯唐的文字没有啥技巧或者修饰。是那种很平实，很真实的文字，就像在读他的日记一样。

从这本书里，可以看到冯唐的出身、经历、困惑，以及的的确确对逝去流年的追忆。对当下老态的觉察，对知天命的豁达，对人世间处处欲望的追寻。

给大脑安装上创新的意识吧

发表于 2021-01-18 分类于 管理类文章

研究有三个层面：顶级的层面，你一手发现了相关/因果规律。这是幸运且令人瞩目的。这种发现可不多。很多比较好发现的规律都已经被人发现并证明过了。留给我们待发现的，隐蔽又难啃；次一点的，是有人提出了相关/因果规律的猜想，但是证明不了，这种情况你可以提出新方法、新范式，并对规律进行证明，这属于第二种创新；再次一点的，是我们绝大多数人使用的创新方法——在已知因果基础上，演绎、发现一点点新的东西。这些创新方法可以是排列组合，可以是跨学科、跨专业解决，总之日进一寸，逐步深入一个小领域。
