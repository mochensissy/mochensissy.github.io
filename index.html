身如浮萍，乱世飘零——《杜甫传》读后感

发表于 2022-01-02 更新于 2022-03-03 分类于 读书笔记

读完此书，花了4h。和狗同学讨论，他说你4h能读出些什么？是一个一个字读的，还是跳读的。我说跳读的，按照主线然后找证据这种方式来读的。基本上了解大概。

那你说说李白和杜甫什么关系？

我说，李白是杜甫的前辈。杜甫眼中，他崇拜且珍视与李白的友谊；可是在李白眼中，似乎没有太care这个朋友。作者洪业说，这是因为，李白是浪漫主义者，不太注重人与人之间的关系；杜甫是现实主义者，对人很重感情。你看看他对对家人、对朋友、对国家的感情，就知道了。

狗同学瞅了我一眼说，我明白了。你就是李白，我就是杜甫。

类比汉、清政治制度看组织设计维度——《中国历代政治得失》读后感

发表于 2022-01-01 更新于 2022-01-23 分类于 读书笔记

去年年末，在几次出差路上断断续续读完，印象尤其深刻，也因这本书认识了钱穆，从此打开了一扇新大门。后来在讲组织设计、组织演变、制度演变时总能时不时想起只字片语。

今年一年，我对组织的理解又有了新的认识和实践。在回京飞机上，我给自己布置了一个任务：从公司组织发展角度，再阅读汉代、清代章节。为何取这两章？其一，此书框架本就通过抽取汉、唐、宋、明、清五个朝代，试图展示中国政治发展演变。我同样在此书基础上继续“抽样”，选取汉、清，首尾两个朝代篇章来读。其二，因为去年已略读过一遍，基本了解作者整体基调和观点；其三，时间有限，基于以上，看两个章节，也基本能形成较完整的看法。

有本事还要会生活——《有本事》读后感

发表于 2021-12-26 更新于 2022-03-03 分类于 读书笔记

从北京飞往海南长达四个多小时，中途打了会儿瞌睡，然后在飞机上读了《有本事》。读到一半，提示要有会员才能继续看。按捺不住看冯唐文字的冲动，文笔轻快让人觉得很轻松。于是落地后买了会员，继续读完。

总体，冯唐的文字没有啥技巧或者修饰。是那种很平实，很真实的文字，就像在读他的日记一样。就是把事情说清楚，很干练的那种感觉。一位男士，如果文字逻辑清晰，想得明白，又能娓娓道来而不刻意，这就是一个才子的标配了。

从这本书里，可以看到冯唐的出身、经历、困惑，以及对对人世间处处欲望的追寻，对当下老态的觉察，对逝去流年的追忆，对天命的豁达。
