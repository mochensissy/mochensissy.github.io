推荐一款马赛克游戏《To The Moon》

发表于 2018-03-31 更新于 2024-12-28 分类于 智慧通达

其实，这个故事的内容，与我的三观并不相合。内容大致是改写记忆，成就梦想。但是《To The Moon》本身讲故事的方式，很值得学习。

故事永远大于文采。即使是平庸的文字，如果有个好剧本，片段下来也不至于太差。正如同这个游戏，马赛克般的像素画面，却给我们呈现了一部感人的故事，配合恰到好处的钢琴曲配乐，催人泪下。故事很像《盗梦空间》和《源代码》的结合。主人公xx博士的工作是帮助临死者实现愿望。她改写临死者的早期记忆，并将新记忆植入，从而让临死者实现愿望。

致逝去的那个时代——《芳华》有感

发表于 2018-03-31 更新于 2024-12-28 分类于 学习札记

《芳华》有模仿《阿甘正传》的痕迹，试图通过一组组小故事串联出时代的变迁。《阿甘正传》里，女主出现篇幅不多，只是每次出现，观众自动脑补她的经历。而《芳华》，它很贪心。同时写了男主和女主两条线，试图在某个时候，通过一种冥冥之中的命运将他们交织在一起。想试图通过两条线，两个小人物，展示时代变迁，这绝非易事。

作者简介

特德·姜，1967年出生于美国，毕业于布朗大学计算机科学系，是美国当代华裔科幻作家。特德·姜的作品不多，自1990年发表处女作《巴比伦塔》至今，总共发表的作品只有八篇，且都是短篇或中篇。他的作品虽少，却几乎篇篇称得上精品。八篇小说让他捧回了包括雨果奖、星云奖、斯特金奖、坎贝尔奖在内的所有科幻大奖的奖杯。

《教育家叔本华》读书笔记

发表于 2018-03-31 更新于 2024-12-28 分类于 学习札记

为了略显高深，很多人都喜欢引用尼采的话。在中国，尼采的名气比在德国更大。事实上，这个19世纪德国哲学家和作家，一生都不得志。45岁时更是因重病病倒。
对于自己的作品，他也是不自信的。在赠书的时候，他甚至是羞愧的他对友人说，你可以从前面读起，也可以从后面读起，千万不要觉得混乱和古怪。如这本书不堪卒读，那里面某些局部或许还是可以看一下的。

奥地利著名作家斯特凡茨威格描述尼采：尼采的悲剧是一出独角戏，在他短暂一生的舞台上，除了他自己外，再没有别的角色……所有的行动全部从他那里发出，又全部返回到他本人……

尼采在21岁读到叔本华著作《作为意欲和表象的世界》，便将之奉为神作。从此以后，对待叔本华的作品，就像儿子在聆听父亲的教诲……

《你一生的故事》：这其实是我们一生的故事

发表于 2017-03-31 更新于 2024-12-28 分类于 学习札记

作者简介

特德·姜，1967年出生于美国，毕业于布朗大学计算机科学系，是美国当代华裔科幻作家。特德·姜的作品不多，自1990年发表处女作《巴比伦塔》至今，总共发表的作品只有八篇，且都是短篇或中篇。他的作品虽少，却几乎篇篇称得上精品。八篇小说让他捧回了包括雨果奖、星云奖、斯特金奖、坎贝尔奖在内的所有科幻大奖的奖杯。

小说简介

该小说选自同名小说《你一生的故事》。这是一部探索语言前世今生的小说。小说分两条线。一条线顺着时间轴叙述科学家对外星人语言研究的全过程。另一条线叙述女儿从生到死的一生，它被碎为一片片拼图小块，散落在小说的各个角落；时空仿佛被打乱，但是越往后读，越是清晰，似乎最终，我们也和主人公一样，学会了外星人的语言。最终看似剪不断理还乱的剧情碎片在读者“习得”外星人语言后被逐渐理清，我们在感受“七肢桶”（文中对外星人的绰号）语言独特魅力的同时，也更感受到了宿命带来的忧伤。
